Common HTML Tags

These are the most commonly used HTML tags and a short description of each.

Block Elements

Block elements take up their own line and take up the full width of the page.

body

body defines the document's body.

div

div defines a section within a document

main

main specifies the main text in a document

footer

footer defines a footer for a document of section

li

li defines a list idem

head

head defines the information about the document

section

section defines a section in a document

article

article defines a article in a document

p

p defines a paragraph in a document or section

html

html defines the root of an HTML document

header

header defines a header for a document or section

nav

nav defines a navigation link

aside

aside defines content aside from the page content

Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

a

This tag definies a hyperlink.

strong

bolds an important text

em

em emphasizes a text

img

img defines an image

span

span defines a section in a document
